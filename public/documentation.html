Introduction

This project aims to build a server-side component for a movies web application. This API helps frontend developers to create functional movie UI for users to read information about different movies and even have a list of their favorite movies.

API Endpoints

This table below provides a list of all current API endpoints with their corresponding description, call URL, HTTP method, data request format, data response format and example responses. This list will continue to be updated as further endpoints are added.

Description Call URL HTTP Method Data Request Format Data Response Format Example Response
Returns list of all movies to user /movies GET Example URL Format:
http://localhost:8080/movies

 JSON 
{
	title: "movie title",
	Author: "movie author",
	Description: "movie description",
	Genre: {
		Name: "genreName",
		Description: "genreDescription",
	},
	Director: {
		Name: "directorName",
		Bio: "directorBio",
	},
	ImageUrl: "movieImage",
	Featured: false,
}
Returns list of all movies genre to users /movies/genre/:genreName GET Example URL Format:
http://localhost:8080/movies/genre/Fantacy

 JSON 
{
	Genre: {
		Name: "genreName",
		Description: "genreDescription",
	},
}
Returns list of all movie directors to users /movies/director/:directorName GET Example URL Format:
http://localhost:8080/movies/director/J.K.%20Rowling

 JSON

Allows users to register /users POST Example URL Format:
http://localhost:8080/users

The JSON format below will be in the body of the POST HTTP request

{
	"name": "Joel",
	"favoriteMovies": "Ready Player One"
}
JSON 
{
    "name": "Joel",
    "favoriteMovies": "Ready Player One",
    "id": "ab55948b-dd77-4aa8-a9e2-ca4879d08bc9"
}
Allows users to update information /users/:id PUT Example URL Format:
http://localhost:8080/users/ab55948b-dd77-4aa8-a9e2-ca4879d08bc9

The JSON format below will be in the body of the PUT HTTP request

{
    "name": "Joel Casey",
    "favoriteMovies": "Ready Player One",
    "id": "ab55948b-dd77-4aa8-a9e2-ca4879d08bc9"
}
JSON 
{
    "name": "Joel Casey",
    "favoriteMovies": "Ready Player One",
    "id": "ab55948b-dd77-4aa8-a9e2-ca4879d08bc9"
}
Allows users to update favorite movies /users/:id/:movieTitle POST Example URL Format:
http://localhost:8080/users/ab55948b-dd77-4aa8-a9e2-ca4879d08bc9/Spirit

 JSON 
{
    "name": "Joel Casey",
    "favoriteMovies": "Spirit",
    "id": "ab55948b-dd77-4aa8-a9e2-ca4879d08bc9"
}
Allows users to delete favorite movies /users/:id/:movieTitle DELETE Example URL Format:
http://localhost:8080/users/ab55948b-dd77-4aa8-a9e2-ca4879d08bc9/Spirit

 TEXT {movieTitle} has been removed from your favorite movies list
Allows users to deregister /users/:id DELETE Example URL Format:
http://localhost:8080/users/ab55948b-dd77-4aa8-a9e2-ca4879d08bc9

 TEXT user with ID {id} has been deleted
